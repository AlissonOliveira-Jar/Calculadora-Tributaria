# 💻 Calculadora Tributária NAF - Frontend

Esta pasta contém a interface do usuário (SPA) da Calculadora Tributária, projetada para o NAF.

## 🛠️ Tecnologias e Bibliotecas
- **Core:** ReactJS e Vite (Node 26 Alpine no Docker).
- **Roteamento:** React Router Dom.
- **Estilização:** Tailwind CSS v4.
- **Exportação:** html2pdf.js.

## 🚀 Execução Local (Desenvolvimento Isolado)

Se preferir rodar ou testar apenas o frontend sem a orquestração da raiz:

### Passo a Passo
1. Garanta que está na pasta correta: `cd frontend`
2. Instale as dependências: `npm install`
3. Inicie o servidor: `npm run dev`

## 📄 Fluxos e Interfaces Implementadas
- Landing Page & FAQ
- Simulador PF (Livro Caixa) e PJ (Simples Nacional)
- Geração de Relatórios PDF
- Telas Mockadas (Login, Cadastro, Fale Conosco)
